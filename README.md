# ai-edu

Codex 교육용 교안 모음입니다. 원본 `yt-assets/ai-productivity` 자료의 형식을 유지하되, 삼성디스플레이 임직원 교육 맥락에 맞춰 주제, 예제 프롬프트, 산출물, 검수 기준을 재작성했습니다.

## 교육 모듈

| 폴더 | 제목 | 핵심 실습 |
|---|---|---|
| `sdc-codex-ppt-proposal/` | Codex 앱으로 삼성디스플레이 업무혁신 제안서 PPT 만들기 | 기준 문서 세팅, Plan 승인, 편집 가능한 PPT 생성, 렌더 검수 |
| `sdc-codex-multithread-newsletter/` | 코덱스로 멀티스레드 활용하기 | 팀장 스레드와 담당자 스레드로 AI 트렌드 뉴스레터 제작 |

## 사용 범위

- 영상 촬영용 자료가 아니라 교육 진행용 교안입니다.
- 예시는 공개 자료 기반 업무 시나리오로 구성했습니다.
- 회사 내부자료, 미공개 기술, 고객 정보, 보안 정보는 실습 데이터로 사용하지 않는 전제를 둡니다.
